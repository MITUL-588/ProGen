## ⚡ Features

- **🚀 Fully Automated:** Generates names, passwords, and temporary emails automatically.
- **📧 Dual Mail Support:** Uses `Mail.tm` by default and auto-falls back to `Temp-mail.io` if needed.
- **🔓 Prime Activation:** Bypasses verification and applies the `ATOMESUS` premium coupon automatically.
- **📦 Auto-Install Dependencies:** Missing modules (like `requests`) are installed automatically in the background.
- **💾 Auto Save:** Successfully created accounts are saved directly into `accounts.txt`.
- **💻 Cross Platform:** Works on Windows PC and Android (Termux).

## 🛠️ Installation & Setup

You can run this tool directly as a `.exe` on Windows or use the Python script on PC/Termux.

### Option 1: Run on Windows (Recommended - No Python Needed)

<a href="https://github.com/MITUL-588/ProGen/releases/latest/download/Progen.exe">
  <img src="https://img.shields.io/badge/Download-Progen.exe-success?style=for-the-badge&logo=windows" alt="Download EXE">
</a>

1. Click the download button above to get the `Progen.exe` file.
2. Double-click to run! All accounts will be saved automatically to an `accounts.txt` file in the same folder.

### Option 2: Run on Android (Termux)

**Step 1:** Clone the repository
```bash
git clone https://github.com/MITUL-588/ProGen.git
cd ProGen
```

**Step 2:** Give executable permission and run the tool
```bash
chmod +x Progen
./Progen
```

## 🚀 Usage Guide

1. Open the tool (`Progen.exe` on Windows or `./ProGen` on Termux).
2. When prompted: `Koyta account lagbe (Enter number of accounts):`, enter how many Prime accounts you want to generate (e.g., `5`).
3. Sit back and relax! The tool will generate the accounts, verify the OTPs, activate Prime, and save the credentials in `accounts.txt`.
